# Archived historical experiment artifacts

Files in this directory preserve earlier research provenance. They were not
generated by the current protocol and must not be presented as protocol-v2.3
evidence.

`legacy_experiments.ipynb` is a byte-identical archival copy of the original
`plots.ipynb` and contains actual earlier experimental work. Some plotting
cells inside the notebook are explicitly labeled simulated; those cells do not
make the notebook itself synthetic.

The executable current research path is `run_triguard.py` plus the numbered
scripts. New protocol-v2.3 figures and tables must be generated from its
content-hashed CSV outputs.
